zerotier

1、在线安装zerotier
curl -s https://install.zerotier.com/ | sudo bash

2、添加开机自启

$ sudo systemctl enable zerotier-one.service

3、启动zerotier-one.service

$ sudo systemctl start zerotier-one.service

4、加入网络

$ sudo zerotier-cli join xxxxxxx
